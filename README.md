# Simple Ball Game using Android Studio [java]

> ## 메인 기능

---

- 핸드폰의 센서를 이용해 기울기로 볼을 움직임
- 30초의 제한 동안 한 레벨 진행
- 30초 안에 볼이 사각형 안에 들어가면 level up
- 들어가지 못하면 life -1
- level up시 볼의 이동 속도 향상, 사각형의 크기 감소
- easy mode / hard mode로 구분해 진행
- 매 레벨 생성되는 사각형의 크기가 볼의 크기보다 작아지면 게임 clear
